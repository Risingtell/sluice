What you are watching. The agent signs an EIP-712 transfer_with_authorization for every tick; the hosted Casper x402 facilitator verifies it and settles a real CEP-18 transfer on casper:casper-test (the facilitator pays the gas). Live on-chain sessions are budgeted per day because the facilitator quota is shared; when the budget is spent the same meter and policy code runs with simulated settlement, and this page says so plainly.
